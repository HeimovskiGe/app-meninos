# 🎮 ROTINACRAFT - Sistema de Rotinas Kids com Tema Minecraft

## 📖 Sobre o Projeto

ROTINACRAFT é um sistema completo de gerenciamento de rotinas para crianças, inspirado no universo do Minecraft. O projeto combina funcionalidades educativas com uma interface divertida e envolvente, incentivando as crianças a completarem suas tarefas diárias através de gamificação.

## ✨ Funcionalidades Principais

### 🏠 Dashboard
- **Visão Geral**: Ranking coletivo e individual
- **Estatísticas Diárias**: Acompanhamento do progresso
- **Sistema de Pontuação**: Baseado em tarefas completadas
- **Níveis e Experiência**: Sistema de progressão gamificado

### 👥 Gestão de Participantes
- **Perfis Personalizados**: Avatares do Minecraft (Steve, Alex, Mobs)
- **Sistema de Níveis**: Baseado na pontuação acumulada
- **Histórico de Tarefas**: Acompanhamento individual
- **Ranking Competitivo**: Comparação entre participantes

### 📋 Sistema de Tarefas
- **Categorias Diversificadas**: Manhã, estudo, higiene, alimentação, etc.
- **Pontuação Variável**: Diferentes valores por tarefa
- **Status de Conclusão**: Controle individual por participante
- **Edição e Remoção**: Gerenciamento completo das tarefas

### 🏆 Sistema de Conquistas
- **Conquistas Temáticas**: Baseadas em categorias de tarefas
- **Progresso Visual**: Barras de progresso animadas
- **Recompensas Especiais**: Bônus por metas atingidas
- **Desbloqueio Gradual**: Sistema de progressão

### ⚒️ Sistema de Crafting
- **Mesa de Trabalho**: Interface estilo Minecraft
- **Receitas Especiais**: Combinações de itens para recompensas
- **Inventário Interativo**: Drag and drop de itens
- **Sistema de Recompensas**: Pontos extras por crafting

### ⚙️ Configurações e Backup
- **Controles de Áudio**: Volume para sons e música
- **Backup e Restauração**: Exportar/importar dados
- **Ações Rápidas**: Completar todas as tarefas, resetar progresso
- **Personalização**: Configurações do sistema

## 🚀 Como Usar

### 1. **Primeiro Acesso**
- Abra o arquivo `index.html` no navegador
- O sistema será inicializado automaticamente
- Adicione participantes através do botão flutuante (+)

### 2. **Adicionando Participantes**
- Clique no botão ➕ flutuante
- Preencha nome e selecione avatar
- Escolha entre Steve, Alex ou mobs especiais
- Confirme para adicionar ao sistema

### 3. **Gerenciando Tarefas**
- Navegue para a aba "TAREFAS"
- Use o botão "Adicionar Tarefa" para criar novas
- Edite ou remova tarefas existentes
- Configure pontos e categorias

### 4. **Completando Tarefas**
- Clique em "Gerenciar Tarefas" no card do participante
- Marque tarefas como concluídas
- Acompanhe o progresso e pontuação
- Veja as conquistas desbloqueadas

### 5. **Sistema de Crafting**
- Acesse a aba "CRAFTING"
- Arraste itens do inventário para a mesa
- Combine itens para criar receitas especiais
- Colete recompensas por crafting bem-sucedido
